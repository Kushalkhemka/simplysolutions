If you are facing issue such as unable to reach windows services/ activation server (Error Codes - 0x80070422, 0x80072ee2, 0x80072EE7 etc.) then you can follow below troubleshooting steps.

1.Search Services on Search Bar on bottom task bar>> Click on Services

2.Find Windows Services - Windows License Manager Service, Windows Update, Windows Event Log, Windows Event Collector, Windows Time.

3.The above all mentioned windows services must be in running mode (status) and startup type should be set to Automatic. If not, then click on the corresponding services and change service status to running and startup type to automatic >> Click on Apply. Now, you can try to activate your windows again.
